An package is the only non ISO 11179 object within the core Aristotle MDR framework. It provides a conveinent system-independant way of collecting similarly related items in a way that allows these packages to be standardised in accordance with the workflow of ISO 11179-3.

Additionally, packages can be used to convieniently group items that need to be standardised together. The ability to cascade a registration status from one item to its child items, means packages can be used to create simple data dictionaries with all subsequent Data Elements and their components standardised in a single action.
